# xgboost_gpt_climate_event_study
A framework for anticipating cascading indirect economic impacts of extreme weather across commodity systems.

GPT_Data_Preprocesser_v2_shyam_updated.ipynb contains the code used to produce the GPT features as indicated in the methodology. XGBoost_Models_Weekly_v2_shyam.ipynb contains the main pipeline used to perform the entire analysis.
